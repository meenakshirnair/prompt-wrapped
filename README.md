# Prompt Wrapped 🎁

See what you actually use AI for. 100% private — your data never leaves your browser.

Upload your ChatGPT or Claude export and get a personalized breakdown of your AI usage patterns.

## Status
🚧 Work in progress. Built in public.

## Tech Stack
React + TypeScript + Vite + Tailwind CSS

## Run Locally
```bash
npm install
npm run dev
```
